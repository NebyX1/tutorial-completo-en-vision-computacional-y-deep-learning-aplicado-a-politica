Paso 0: (Opcional) Clonar Repositorio
>git clone https://github.com/NebyX1/tutorial-completo-en-vision-computacional-y-deep-learning-aplicado-a-politica

Paso 1: Crear el entorno virutal en la carpeta del proyecto:
>python -m virtualenv venv

Paso 2: Para activar el entorno virtual para empezar a trabajar, solo hay que escribir en la carpeta raíz:
>venv\Scripts\activate

Paso 3: Instalar todas las librerías y dependencias que usaremos dentro de nuestro entorno virtual
Escribir el siguiente comando: 
>pip install -r requirements.txt

Paso 4: para ejecutar cada uno de los scripts hay que poner "python" primero:
>python us_train_model.py
>python us_test.py
>python us_presidents_model.py
>python us_average.py

>python uy_emotion_analyzer.py
>python uy_average.py
